## Tech info

- The Vinyl shop website was written in **React**. I used **SCSS** and split the code into
  modules.
- In a project, I wanted to learn content flow and .

## Technologies

- Javascript
- React
- Context API
- Sassy CSS

## Setup

To run this project, install it locally using npm:

```sh
$ npm install
$ npm start
```
